# Physis Techne Symposium Hackathon - 2026

### Team Name: Reactive Power


This repository contains all the code files and the report for the submission of the Electronics Problem Statement of this hackathon organized by Dept. of Physics, IIT (ISM) Dhanbad, by the team Reactive Power.

#### Please note that all the explanations and the plots are present in the [Mission Report](https://github.com/legend27th/Reactive_power/blob/main/Mission_Report.pdf).

Also, the codes are given in the GitHub Repository files as Stage 0, 1, 2, 3 & 4 and the ``Extras_QEC.py`` contains the Quantum Error Correction style approach to increase the accuracy of Data Acquisition.

The extracted bits are there in ``Decoded_Bits.npy``.
